# 🌦️ Weather App

## Description  
This project is a weather application built using React that allows users to search for the current weather in any city. The app takes user input and fetches real-time weather data from a public API, displaying information such as temperature, weather conditions, and location details. I built this project to practice working with APIs and handling user input in a React application.  

To create this app, I used React for the frontend and integrated a weather API to retrieve live data. I also used JavaScript for logic, CSS for styling, and GitHub for version control. One challenge I faced was handling errors when users entered invalid city names, but I fixed this by adding proper error handling and conditional rendering. I also worked on improving the UI to make it simple and easy to use. Overall, this project helped me better understand API calls, state management, and building interactive applications.

## 🛠️ Technologies Used  
- React  
- JavaScript  
- HTML  
- CSS  
- Weather API
